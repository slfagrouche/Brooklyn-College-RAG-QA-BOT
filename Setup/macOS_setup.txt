1. Install Homebrew:
    Documentation found at: https://brew.sh/

2. Install pyenv:
    xcode-select --install
    brew install openssl readline sqlite3 xz zlib
    brew update
    brew install pyenv

3. Add pyenv to your $PATH (if you are not using zsh as a shell, you have to change ~/.zshrc accordingly):
    echo 'eval "$(pyenv init --path)"' >> ~/.zshrc

4. Install Python Version 3.11.0:
    pyenv install 3.11.0

5. Clone your repository:
    git clone https://huggingface.co/spaces/Slfagrouche/Brooklyn-College-RAG-QA-BOT

6. Navigate to the cloned repository folder:
    cd Brooklyn-College-RAG-QA-BOT

7. Use the installed Python version in the cloned repository folder:
    pyenv local 3.11.0

8. Create virtual environment in the cloned repository folder:
    python -m venv .brooklyn-college-rag-qa-bot-venv

9. Activate the virtual environment:
    source .brooklyn-college-rag-qa-bot-venv/bin/activate

10. Install the dependencies listed in the requirements.txt file:
    pip install -r requirements.txt

11. Run your Gradio application:
    python your_gradio_script.py
